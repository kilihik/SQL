# 1. Ресурсы для генерации файлов изменения размера

Random Var

Ссылка: https://randvar.com/generator/file

Описание: Позволяет генерировать файлы разных размеров и форматов. Очень гибкий в настройках.

FileMock

Ссылка: https://filemock.com/

Описание: Простой и понятный генератор. Позволяет генерировать видео, аудио, изображение, документы, после чего скачиваете файл с заданными параметрами.

# 2. Ресурсы для формирования тестовых данных (ФИО, телефон, email, адрес)
Эти сервисы генерируют правдоподобные данные, соответствующие стандартам выбранной страны.

Generate test data. Quickly

Ссылка: https://generatedata.com/

Описание: Простой в использовании интерфейс. Предварительный просмотр того, что вы создаете, во время сборки. Более 30 типов данных для генерации (имена, адреса электронной почты, страны и т. д.). Более 10 форматов генерации (JSON, CSV, XML, SQL и т. д.). Предоставляет взаимосвязанные данные (например, по стране, региону, городу). Сохраните свои наборы данных (требуется учетная запись пользователя)

Random Data Tools

Ссылка: https://randomdatatools.ru/

Описание: Программа генерирует случайное имя, фамилию, отчество, контактные, паспортные и другие данные для дальнейшей работы с ними. Например, для заполнения базы данных тестовыми данными. Имеет удобный интерфейс для быстрого копирования информации. Содержит в себе более 250 имен и более 3500 фамилий, из которых способна собрать до 200 миллионов уникальных персонажей и способна автоматически вносить их в таблицу вашей базы данных. Также есть функции генерации случайных чисел и хеширования алгоритмами MD5 и SHA. Для разработчиков предоставлено расширенное API.
