|------- | :-------:| :-------:|:-------:|

|Ввод пароля |	Клик вне поля ввода |	Страница авторизации |	Деактивация поля ввода |
|Ввод пароля  |	Клик в поле логина |	Ввод логина |	Возврат к редактированию логина |
|Ввод пароля |	Нажатие кнопки "Войти" |	Проверка данных |	Отправка данных на сервер |

|Ошибка авторизации |	Исправить логин |	Ввод логина |	Редактирование поля логина |

# Диаграмма состояний для страницы авторизации

```mermaid
stateDiagram-v2
    [*] --> Страница_авторизации
    Страница_авторизации --> Ввод_логина : Клик в поле логина
    Страница_авторизации --> Ввод_пароля : Клик в поле пароля
    Страница_авторизации --> Проверка_данных : Нажать "Войти"
    
    Ввод_логина --> Страница_авторизации : Клик вне поля
    Ввод_логина --> Ввод_пароля : Tab
    Ввод_логина --> Проверка_данных : Нажать "Войти"
    
    Ввод_пароля --> Страница_авторизации : Клик вне поля
    Ввод_пароля --> Ввод_логина : Клик в логин
    Ввод_пароля --> Проверка_данных : Нажать "Войти"
    
    Проверка_данных --> Успешная_авторизация : Данные верны
    Проверка_данных --> Ошибка_авторизации : Неверные данные
    Проверка_данных --> Ошибка_сети : Нет связи
    
    Ошибка_авторизации --> Ввод_логина : Исправить логин
    Ошибка_авторизации --> Ввод_пароля : Исправить пароль
    Ошибка_авторизации --> Страница_авторизации : Закрыть ошибку
    
    Ошибка_сети --> Страница_авторизации : Повторить
    Успешная_авторизация --> [*]
```
