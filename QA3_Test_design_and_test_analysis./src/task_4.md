#  Анализ анкеты для оценки кредитоспособности
---
## Количество всех возможных комбинаций
## Для расчета общего количества комбинаций перемножим количество вариантов по каждому параметру:

- Возраст: 3 варианта (18–35, 36–55, 56–75)
- Наличие задолженностей: 2 варианта (да, нет)
- Тип трудовых отношений: 3 варианта (ИП, Самозанятый, Бессрочный договор)
- Наличие действующих кредитов: 2 варианта (да, нет)
Общее количество комбинаций: 3 × 2 × 3 × 2 = 36 уникальных комбинаций

Проверка полноты и корректности набора уникальных пар
Методы проверки полноты:
Матрица покрытия пар - создаем таблицу, где отслеживаем все возможные парные комбинации:

- Возраст × Задолженности (3×2 = 6 комбинаций)
- Возраст × Трудовые отношения (3×3 = 9 комбинаций)
- Возраст × Действующие кредиты (3×2 = 6 комбинаций)
- Задолженности × Трудовые отношения (2×3 = 6 комбинаций)
- Задолженности × Действующие кредиты (2×2 = 4 комбинации)
- Трудовые отношения × Действующие кредиты (3×2 = 6 комбинаций)

Автоматизированная проверка - использование специализированных инструментов для pairwise testing:
- PICT (Microsoft)
- Hexawise
- Pairwise.org
Статистический анализ - проверка распределения значений каждого параметра:
Каждый вариант возраста должен встречаться примерно одинаковое количество раз
Аналогично для других параметров
## Методы проверки корректности:
### Валидация граничных значений - убедиться, что учтены крайние случаи:
- Минимальный возраст (18–35) с различными комбинациями других параметров
- Максимальный возраст (56–75) с различными комбинациями
### Проверка противоречивых комбинаций - выявление логически несовместимых пар:
Например, проверка комбинаций с одновременным наличием задолженностей и отсутствием кредитов
Экспертная оценка - анализ тестовых случаев бизнес-аналитиком или кредитным экспертом на предмет реалистичности и полноты покрытия бизнес-сценариев
### Критерии успешной проверки:
Все 37 парных комбинаций должны быть покрыты хотя бы один раз
Каждый параметр должен встречаться в различных контекстах
Набор должен быть минимальным, но достаточным для выявления основных дефектов
