# Техническое задание на приложение “Найди и иди”:

**ID требования** | **Описание требования**
---|---
SC21-01 | **Поле ввода названия магазина**<br>- Поле ввода названия магазина принимает любые символы.<br>- Поле содержит плейсхолдер.<br>- Плейсхолдер исчезает при вводе символов.<br>- Минимальное количество символов — 2.<br>- В поле сохраняются результаты предыдущих поисков.
SC21-02 | **Поле выбора типа магазина**<br>- Поле выбора типа магазина представляет собой выпадающее меню со следующими вариантами выбора значений:<br>1. «Продуктовые магазины»;<br>2. «Магазины стройматериалов»;<br>3. «Аптеки»;<br>и т.п.<br>- Поле содержит плейсхолдер «Выберите тип магазина».<br>- При наличии выбранного значения оно отображается в поле.
SC21-03 | **Кнопка «Поиск»**<br>- Кнопка «Поиск» активна только в случае заполнения минимум одного поля (некликабельна при отсутствии заполнения обоих полей).<br>- Успешные результаты поиска отображаются в виде флажков на карте.<br>- Карту можно скроллить и зуммировать (стандартные жесты).
SC21-04 | **Кнопка «Построить маршрут»**<br>- Кнопка «Построить маршрут» активна в следующих случаях:<br>1. Найден результат поиска введенного пользователем названия магазина и выбран его флаг на карте.<br>2. Выбран тип магазина (маршрут будет построен до ближайшего магазина с указанным типом).<br>- Кнопка содержит иконку в виде изогнутой стрелки, надпись «Построить маршрут» и цвет фона #fff (белый).<br>- При нажатии на кнопку цвет фона меняется на #586d73 (серый) и название кнопки меняется с «Построить маршрут» на «Отменить маршрут».
SC21-05 | **Область карты**<br>- Область карты занимает половину области экрана телефона.<br>- Для построения маршрута требуется выполнение условий в требовании SC21-04.<br>- Построенный маршрут можно отменить только нажатием кнопки «Отмена».
SC21-06 | **Общие требования к работе приложения**<br>- При повторном открытии приложения отображать незаконченный ранее маршрут.<br>- Отключать точную геопозицию при низком заряде аккумулятора.

## Выявленные ошибки в технической документации:

### Ошибка: Противоречие в описании отмены маршрута

**Фраза:** 
- SC21-04: "При нажатии на кнопку цвет фона меняется на #586d73 (серый) и название кнопки меняется с «Построить маршрут» на «Отменить маршрут»."
- SC21-05: "Построенный маршрут можно отменить только нажатием кнопки «Отмена»."

**Критерий ошибки:** Противоречивость

**Пояснение:**
1. В требовании SC21-04 указано, что кнопка меняет название на «Отменить маршрут»
2. В требовании SC21-05 указано, что для отмены маршрута используется кнопка «Отмена»
3. Это создает неоднозначность в интерфейсе - непонятно, какая именно кнопка используется для отмены маршрута
4. Разработчики могут реализовать разные варианты, что приведет к несогласованности интерфейса

### Ошибка: Неопределенность в условиях активации кнопки "Построить маршрут"

**Фраза:**
"Кнопка «Построить маршрут» активна в следующих случаях:
1. Найден результат поиска введенного пользователем названия магазина и выбран его флаг на карте.
2. Выбран тип магазина (маршрут будет построен до ближайшего магазина с указанным типом)."

**Критерий ошибки:** Неоднозначность

**Пояснение:**
1. Не указано, должна ли кнопка быть активной при одновременном выполнении обоих условий
2. Не определен приоритет условий - что имеет больший вес при конфликте условий
3. Неясно, как система определяет "ближайший магазин" при выборе типа
4. Отсутствуют критерии определения "ближайшего" магазина

### Ошибка: Неполное описание поведения при поиске

**Фраза:**
"Успешные результаты поиска отображаются в виде флажков на карте."

**Критерий ошибки:** Неполнота

**Пояснение:**
1. Не описано поведение при отсутствии результатов поиска
2. Не указано, что происходит с предыдущими результатами при новом поиске
3. Не определено максимальное количество отображаемых флажков
4. Не описана обработка ошибок при поиске (нет сети, ошибки сервера и т.д.)

### Ошибка: Противоречие в цветовых обозначениях

**Фраза:**
"цвет фона меняется на #586d73 (серый)"

**Критерий ошибки:** Неточность

**Пояснение:**
1. Цвет #586d73 в HEX-формате является темно-сине-зеленым, а не серым
2. Это может привести к неправильной визуальной реализации интерфейса
3. Для серого цвета следует использовать значения с одинаковыми компонентами RGB (например, #808080)

### Ошибка: Неопределенность в сохранении состояния

**Фраза:**
"При повторном открытии приложения отображать незаконченный ранее маршрут."

**Критерий ошибки:** Неоднозначность

**Пояснение:**
1. Не определено, что считается "незаконченным" маршрутом
2. Не указаны временные рамки - как долго сохраняется состояние маршрута
3. Не описано поведение при множественных незаконченных маршрутах
4. Не определено, сохраняется ли состояние при принудительном закрытии приложения

### Ошибка: Неполное описание работы с геолокацией

**Фраза:**
1. "Отключать точную геопозицию при низком заряде аккумулятора."
2. "Геолокация всегда разрешена приложению." (Описанов условии)
**Критерий ошибки:** Неполнота

**Пояснение:**
1. Не определено, что считается "низким зарядом аккумулятора" (процентное значение)
2. Не описано, какая геолокация используется вместо точной (приблизительная или отключается полностью)
3. Не указано, уведомляется ли пользователь об этом изменении
4. Не определено, как это влияет на построение маршрутов
